# shuanq.top 落地页需求分析

## 一句话定义

一个"集体栓Q"病毒式落地页——访客打开看到滚动的栓Q语录和全站计数器，可以一键提交自己的栓Q瞬间。目标是好笑到截图转发。

## 核心目标

- 用户行为：打开 → 哈哈一笑 → 截图/转发给朋友
- 成功指标：分享率（用户看完后发给别人的比例）

## 功能清单

### P0（必须有）

1. **全站栓Q计数器** — 页面顶部/醒目位置，显示"已有 XX 人在此栓Q"
2. **滚动语录墙** — 弹幕式/列表式滚动展示栓Q语录
   - 预设语录：上线时内置 50+ 条精选内容，保证冷启动体验
   - 真实提交：用户可以提交自己的语录，混入滚动流
3. **一键提交** — 极简输入框，写一句话 + 提交，无需登录
4. **移动端优先** — 主要传播渠道是微信/社交媒体，必须移动端体验优秀

### P1（第二阶段）

- 关键词过滤（基础内容审核）
- 热门语录排行
- 分享卡片生成（带 shuanq.top 水印的截图）

### P2（有流量后）

- 实时推送（WebSocket）
- 每日栓Q指数
- 语录点赞/投票

## 技术方案

```
架构: 静态前端 + Serverless API + Turso (LibSQL)
部署: Vercel / Cloudflare Pages
数据库: Turso (libsql://shuanq-mayf.aws-us-west-2.turso.io)

前端: Astro 或 Next.js（静态生成 + 客户端交互）
API: Serverless Functions（提交语录、获取语录列表、计数）
存储: Turso — quotes 表（id, content, created_at, approved）
```

### 数据模型

```sql
CREATE TABLE quotes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  content TEXT NOT NULL,
  ip_hash TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  approved INTEGER DEFAULT 1
);

CREATE TABLE counter (
  id INTEGER PRIMARY KEY,
  count INTEGER DEFAULT 0
);
```

### API 端点

- `GET /api/quotes` — 返回最新 N 条语录（预设 + 已审核的用户提交）
- `POST /api/quotes` — 提交新语录（限流：同 IP 每分钟 1 条）
- `GET /api/count` — 获取总计数
- `POST /api/shuanq` — "我也栓Q"点击，计数 +1

## 设计方向

### 氛围关键词

- 荒诞、欢乐、集体感、网络梗文化
- 视觉冲击 > 精致排版
- 大字、动感、密集信息流

### 页面结构（建议）

```
┌─────────────────────────────┐
│  🔢 已有 12,847 人在此栓Q    │  ← 大号计数器，数字动态跳动
├─────────────────────────────┤
│                             │
│  ← 语录弹幕区 →             │  ← 占据主视觉，滚动/飘动
│  "今天甲方改了第8版，栓Q"    │
│  "考研二战失败，栓Q"         │
│         ...                 │
│                             │
├─────────────────────────────┤
│  [ 输入你的栓Q瞬间... ]  📤  │  ← 底部固定输入框
└─────────────────────────────┘
```

### 色彩倾向

- 主色调偏暗或饱和度高（弹幕类页面深色背景效果好）
- 文字醒目、对比度强
- 可以带点"赛博朋克"或"抽象风"

## 约束与风险

| 风险 | 应对 |
|------|------|
| 冷启动内容少 | 预设 50+ 条语录，上线即有氛围 |
| 恶意内容 | P0 阶段先人工审核 + 基础关键词过滤 |
| 流量突增 | Serverless + Turso 天然弹性 |
| 抄袭/竞品 | 先发优势 + 域名本身就是壁垒 |

## 验收标准

1. 移动端打开 < 2 秒可见完整页面
2. 语录滚动流畅，视觉有冲击力
3. 用户 3 步内完成提交（点输入框 → 打字 → 发送）
4. 预设语录足够有趣，冷启动不尴尬
5. 截图后域名 shuanq.top 清晰可见（传播时自带推广）
